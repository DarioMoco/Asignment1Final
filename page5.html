Project idea:

Overview:

As a project I would like to develop light, fun application which will connect colour to a melody or tone. I would like to develop it as the computer application. This application would be in form of question and answer.

Question would be asked to user and then the answer should be picking up a colour evoked by that question. Once when 4 questions answered a 4 chord progression should be calculated and played either as a arpeggio or a chord melody formation. Choose of colour would be driven by questions from motivation psychology and also connected to a harmonized chords from one scale so it makes musical sense once when any colour chosen.

Motivation:

This application should be executed as a light fun application for relaxing. A lot of people would download this and have fun. Musicians would be able to use it to generate chord progressions if they lack ideas but in a fun way. Another interesting thing is this doesn’t necessarily have to be executed as a psychological motivation questionary. It could be a version for zodiac lovers to for example do their perfect match of zodiac signs in accordance to behaviour or colour or felling and create music that way. For marketing purposes colours could be changed by matching zodiac signs, or signs with colours and hearing the melody.

Description:

As described above application meant to be started with question and then answers should be colours picked bby user. Once answers are assembled for three or four questions melody should be calculated out of colours chosen. Key should be determined by the first and/or second colour.

Form of questions could vary from various topics. The most interesting topic in my opinion are the questions from motivational psychology. The questions should be assembled in maybe 5 different types (different sets randomly chosen at the app start) at the beginning to keep app easy and simple.

When first colour taken it should chose the key and then the rest of the colours/chords should be formed as part of that key. It would be better if possible through development, to determine the key after two answers or even three, to give it more variety of melodies.

Example:

Chose the colour your day started with? Colour Blue key chosen C Major.

Chose the colour your day went so far? Brown. If that note doesn’t belong to a key of C if its G# for example then new combination should be chosen wheere this color belongs to key of C (Mathematically do all the combinations and make it working-How each colour acts in a constellation of other colour being chosen as a key.)

Chose the colour you hope how it will end? Yellow, same process as above,with narowing number of possible chord combos if there is more questions.

Three chords melody should be played and instruments should be available. Violin, piano, cello, etc

Compose Colours!!

This button should pop up for execution of chord progression

The melody could be played as an arpeggio or just chord pattern and it should be possible to change after answers are given and melody established. Also play on different instruments like guitar violin piano or cello should be available

Environment should be pleasing with calm background photos of nature filled with motivational famous quotes.

This interactive programme can be fun to describe the felling by colour and then hear it as a sound or melody. Another level of application could be a shades of certain colour nominated as a chords within harmonised scale. Different shades of same colour or even different colours and their shades would form melody of that combination. This would be probably bigger project since shades of one colour chosen but in this way variety of application versions could be widen to almost any topic to be connected into this idea.

Some serious application of this software would be using it for therapeutic purposes for people with disabilities or people negatively affected by life. During my previous work with kids affected by war I realised that connection to music and evoking feelings through colour and music can be extremely positive and can give great results.

There is a lot of chord progression generator websites or applications but none of them which can be used in a fun simple way for relaxation as well as for other more serious purposes.
